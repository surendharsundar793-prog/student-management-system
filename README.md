Student Management System using Java & JDBC

🚀project Description

Developed a console-based Student Management System using Java and JDBC.

The application performs CRUD (Create, Read, Update, Delete) operations on student records.

Integrated MySQL database for data storage and retrieval.

Used PreparedStatement to prevent SQL Injection.

 Technologies Used

1.Java

2.JDBC

3.Postgresql

4.Eclipse IDE

⚙Features

1.Add new student details

2.View all students

3.Update student information

4.Delete student records

5.Search student by ID
